## Project Structure

* `task_1.py`: A calculator script that performs basic arithmetic.
* `task_2.py`: A personalized greeting script.

## Functionality

### Task 1: Basic Arithmetic Calculator
This program takes two numeric inputs from the user and performs the following operations:
* **Addition** ($+$)
* **Subtraction** ($-$)
* **Multiplication** ($*$)
* **Division** ($/$)



### Task 2: Personalized Greeting
This script collects the user's first and last name separately and prints a formatted welcome message. It demonstrates:
* Using the `input()` function.
* String handling and output formatting.
